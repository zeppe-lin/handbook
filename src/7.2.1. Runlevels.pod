=head3 B<7.2.1. Runlevels>

The following runlevels are used in Zeppe-Lin (defined in
F</etc/inittab>):

    +----------+------------------+
    | Runlevel | Description      |
    +----------+------------------+
    | 0        | Halt             |
    | 1 (S)    | Single-User Mode |
    | 2        | Multi-User Mode  |
    | 3-5      | (Not Used)       |
    | 6        | Reboot           |
    +----------+------------------+

See L<inittab(5)> for more information about runlevels.

=cut
